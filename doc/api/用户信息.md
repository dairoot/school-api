## 用户信息接口: `get_info`

* **请求示例**

```python
from school_api import SchoolClient
school = SchoolClient('http://210.38.137.126:8016')
student = school.user_login('xxxxxxxx', 'xxxx')
# 获取用户信息
student.get_info()
```

* **返回数据格式**

```json
{   
    "real_name": "姓名",
    "birth_date": "出生日期",
    "id_card": "身份证",
    "enrol_time": "入学日期",
    "grade": "年级",
    "hometown": "省份",
    "sex": "性别",
    "faculty": "院系部",
    "specialty": "专业",
    "class_name": "班级",
    "education_level": "学历",
    "eductional_systme": "学制"
}
```
